A small client application that displays a list of authors allows you to edit and add them.

The author has the following attributes:

Surname, Name, Patronymic - text fields (all fields are required);
Date of birth (calendar must be used to select the date);
List of books - each book has its own attributes (name, genre, number of pages);
It is possible to edit the data of the author, the ability to add, delete and edit books of an individual author.

The ability to search for books by name.

The data is stored in Local Storage.

HTML5
CSS3
